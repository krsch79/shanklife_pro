# Shanklife Pro

Hovedapplikasjonen i Shanklife-økosystemet, inkludert delte brukere, GolfBox/BalleTour-funksjoner og sideapper. Se `AGENTS.md`, changelog-filer, `deploy/` og `scripts/` for drift. Installer `requirements.txt`; `.env`, databaser, tokens og backuper må gjenopprettes separat og skal aldri committes.
